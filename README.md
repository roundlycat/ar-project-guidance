# AR Project Guidance System

**Augmented Reality interface for hands-free project assistance**

An Android AR application that provides contextual, spatial guidance for technical projects - from breadboard wiring to equipment assembly - using AR anchors, spatial audio, and gesture control.

## Project Vision

Traditional technical documentation requires constantly switching between instructions and hands-on work. This AR system overlays guidance directly onto your physical workspace, enabling:

- **Spatial Instructions** - See step-by-step guidance anchored to physical objects
- **Hands-Free Interaction** - Voice commands and gesture control
- **Contextual Awareness** - Recognizes components and adapts guidance
- **Progress Tracking** - Visual confirmation of completed steps
- **Multi-Modal Guidance** - Visual overlays + spatial audio + haptic feedback
